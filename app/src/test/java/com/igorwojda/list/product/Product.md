# Product

## Instructions

Given list of integers and positive integer `n` implement a function which returns product of all numbers in the list.

[Puzzle](Product.kt) | [Solution](ProductSolution.kt)

## Examples

```
product(1, 2) // 2

product(1, 2, 4) // 8

product(2, 4, 10) // 80
```
