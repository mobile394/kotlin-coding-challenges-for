# Print numbers

## Instructions

[Puzzle files](.)

Write a function that accepts a positive Int `n`. The function should return a list representing all numbers from `n` to `1`. If `n` value
is zero then empty list should be returned.

## Examples

```
printNumber(0) // []
printNumber(2) // [2, 1]
printNumber(5) // [5, 4, 3, 2, 1]
```
